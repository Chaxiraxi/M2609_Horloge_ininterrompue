Construct the UI controller.

Description: Stores hardware references and initializes controller state for LCD rendering, source selection, and manual time configuration.

Parameters
lcdReference to the 8×2 LiquidCrystal display.
setButtonReference to the SET GPIO button.
coordinatorReference to the central time coordinator.
sourcesArray of source pointers (same order as coordinator).
sourceCountNumber of sources.
mcpIoIoAbstractionRef for the MCP23017 at address 0x20.
loggerOptional logger for debug/warning messages.
