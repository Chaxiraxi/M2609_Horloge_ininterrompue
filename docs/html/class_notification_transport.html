Detailed Description

Abstract transport interface for notifications.

Description: Defines the contract used by Notification to forward messages to one or more output channels (e.g., Serial, LCD, web endpoint).

Implement this interface to route notifications to another sink (HTTP, file, display, etc.). Implementations should be lightweight and non-blocking where possible for embedded use.
