Detailed Description

Abstract base class for a time/date provider.

Description: Defines a common interface for time sources (e.g., GPS, DAB). A source can be enabled/disabled, and can provide time, date, or date-time values through a consistent API.

Inputs
Derived classes provide the hardware/library dependencies (e.g., references to GPS/DAB objects).
Outputs
Implementations return true when valid data is available, false otherwise.
