Detailed Description

Time source implementation backed by a DAB (Digital Audio Broadcasting) receiver.

Description: Provides date/time values retrieved from a DABShield instance. Applies a fixed timezone offset to the hour field (hour wrapping only).

Inputs
Requires references to an initialized DAB object, a DABTime storage struct, and a shared hasService flag.
Outputs
getDateTime/getTime/getDate return true when DAB time is available and the source is enabled.
